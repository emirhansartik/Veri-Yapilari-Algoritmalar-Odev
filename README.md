# Veri Yapıları ve Algoritmalar Proje Ödevi

Bu repo, Veri Yapıları dersi kapsamında hazırlanan algoritmaları içermektedir.

## İçerik
1. **Soru 1:** Insertion Sort ve Selection Sort algoritmalarının 500 rastgele sayı üzerindeki çalışma sürelerinin karşılaştırılması.
2. **Soru 2:** Bir diziyi "En Büyük - En Küçük" sırasıyla (80, 3, 60, 9...) düzenleyen özel algoritma tasarımı.
3. **Soru 3:** Trie (Metin Ağacı) veri yapısının mantığı ve C dili ile gerçekleştirimi.
